# Scheduler

## Основные операции

- создать задачу
- обновить задачу
- удалить задачу
- включить/выключить задачу
- посмотреть ближайшие запуски

## Триггеры

- `once` — разовый запуск (`datetime`)
- `daily` — ежедневно (`time`)
- `weekly` — по дням недели (`time`, `day_of_week`)
- `interval` — интервал (`hours`/`minutes`)
- `cron` — cron (`cron_expression`)
