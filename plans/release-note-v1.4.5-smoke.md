### Быстрый запуск авто-прогона

```bash
uv run python scripts/api_smoke_run.py \
  --base-url http://127.0.0.1:5010 \
  --api-key <TOKEN> \
  --duration-minutes 30 \
  --interval-seconds 15
```

Скрипт отправляет циклические `GET /health`, `GET /api/v1/status`,
`GET /api/v1/config` и дописывает итоговый блок в этот файл.

## Чеклист

- [ ] Запущен GUI (`uv run python main.py --gui`).
- [ ] На вкладке `API` сервер запущен из GUI.
- [ ] Включено live-обновление окна логов.
- [ ] В течение `30+` минут выполнялись периодические запросы:
  - [ ] `GET /health`
  - [ ] `GET /api/v1/status`
  - [ ] `GET /api/v1/config`
- [ ] Выполнен перезапуск API из GUI в ходе прогона.
- [ ] Окно логов не зависает и не теряет новые строки.
- [ ] В `logs/api/` пишется актуальный файл `api_YYYY-MM-DD.log`.
- [ ] Ошибок уровня `ERROR`/`CRITICAL`, связанных с логами, не обнаружено.
