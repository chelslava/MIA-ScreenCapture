# Logs and Diagnostics

## Где лежат логи

- Общие: `logs/`
- API: `logs/api/api_YYYY-MM-DD.log`

## Что логируется

- запуск/остановка API
- access-логи запросов (`method`, `path`, `status`, `latency`, `request_id`, `ip`)
- ошибки записи/кодирования

## Диагностика

В GUI используйте вкладку `Диагностика`.
