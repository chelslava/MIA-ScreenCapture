## Быстрый Авто-Прогон (Опционально)

```bash
uv run python scripts/api_smoke_run.py \
  --base-url http://127.0.0.1:5010 \
  --api-key <TOKEN> \
  --duration-minutes 30 \
  --interval-seconds 15
```
